1.1 Descrición

Título: Acordo.

A aplicación é un CRUD desenvolvido en Java usando Swing con implementación JPA Hibernate e base de datos MySQL. Trátase dunha aplicación de escritorio de apuntamento de notas e recordatorios.

Toda a aplicación consta de dúas pantallas, unha pantalla de lista de tarefas e unha pantalla para rexistrar e editar tarefas.

1.2 Funcionalidades

Pantalla de To Do:

 

  • Botón «Novo» para gardar novos rexistros

  • Botón «Editar» para a edición de rexistros existentes.

  • Botón «Eliminar» para a eliminación de rexistros existentes.

  • Botón «Listar» para a listaxe de rexistros gardados.

 

A pantalla actualiza automaticamente a lista de rexistros de todas as operacións e mostra mensaxes de confirmación únicas para cada tipo de operación realizada.

 

Pantalla principal:

  • Botón «Gardar» para gardar un novo rexistro ou un rexistro editado.

  • Botón «Cancelar» para cancelar a operación.

 

A pantalla dispón de validación de campos obrigatorios, non permitindo gardar un rexistro sen conter a información obrigatoria, cando se intente gardar un rexistro coa información que falta, o campo obrigatorio marcarase cun bordo vermello e un globo de mensaxes informando do cumprimento obrigatorio de o campo en cuestión.

 

Datos das tarefas/notas:

 

  • ID* (xerado automaticamente para cada nova tarefa creada).

  • Título*.

  • Estado* (Completado/Non completado).

  • Descrición.

  • Data de creación (xerada automaticamente ao engadir un novo rexistro).

  • Editar data (xerado automaticamente ao editar un rexistro existente).

  • Data de finalización (xerado automaticamente ao marcar a tarefa como completada).

 

Todos os campos marcados con un * significa que son obrigatorios.
